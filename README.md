## 学习通智能辅助答题系统 - 使用手册

**作者：Bart**

本工具旨在通过豆包 AI 实现学习通网页端的自动化答题与简答录入，帮助你更高效地处理任务。

### 1. 准备工作（必备环境）

在安装脚本前，请确保浏览器已安装以下插件：
* **学习通强制粘贴纯文本助手**:这个可以辅助粘贴文本到文本框内,平时也可单独用,目前问题比较少,强力推荐.
* **Tampermonkey (油猴插件)**：这是运行脚本的核心引擎。
* **Disable Content-Security-Policy (CSP 禁用插件)**：**（重要）** 学习通和豆包的网页安全策略会拦截脚本的跨域操作及第三方库加载。如果不禁用 CSP，截图功能和自动通信将无法工作。

> **关于 CSP 插件的说明：**
> 请前往 Chrome/Edge 扩展商店搜索并安装 `Disable Content-Security-Policy`。安装后，请确保该插件图标处于**激活（Enabled）**状态。它能确保 `html2canvas` 脚本顺利加载，并允许网页间的必要数据交互。

### 2. 脚本安装

1. 打开你的 Tampermonkey 仪表盘。
2. 点击“添加新脚本”。
3. 将代码完全复制粘贴进去并保存（Ctrl+S）。

### 3. 使用流程

安装完成后，请按照以下步骤操作：

**第一步：页面绑定**

1. 在两个标签页分别打开**学习通答题页面**和**豆包聊天页面**。
2. 在豆包页面的右下角，你会看到一个“绑定”按钮，点击它，系统会自动将当前页面与学习通脚本绑定（只需绑定一次，刷新有效）。

**第二步：初始化**

1. 回到学习通答题页面。
2. 在悬浮窗上点击“初始化规则”。此时豆包页面会自动输入一段特定的指令，告诉 AI 如何输出 JSON 格式的答案。

**第三步：答题**

* **发送文字**：点击“发送文字”，脚本会自动提取页面题目文字并发送给豆包。
* **发送截图**：点击“发送截图”，脚本会自动截取题目区域并以图片形式发送。
* **自动处理**：等待 AI 回复，脚本会自动捕捉 JSON 数据并将答案填入对应的选项框或简答框中。

### 4. 常见问题排查 (Q&A)

* **Q: 点击“发送截图”后，悬浮窗提示“截图处理中”，但一直没反应？**
* **A**: 绝大多数情况是因为 CSP 拦截了截图库。请检查浏览器右上角的 `Disable Content-Security-Policy` 插件是否处于开启状态。


* **Q: 答案没有自动填入？**
* **A**: 检查豆包是否回复了正确的 JSON 格式。如果 AI 输出了太多的废话，建议重新点击“初始化规则”发送一遍指令。


* **Q: 页面跳转后脚本失效了？**
* **A**: 本脚本已内置 SPA 监听，通常会自动重载。如果确实失效，手动刷新页面即可。
