# 学习通联动豆包自动答题脚本
作者：Bart
---
借助豆包AI实现学习通网页端自动答题，支持文字读题、图片识图作答，自带手动刷新答案、难题自动跳题功能

## 功能更新
1. 增设手动抓取按钮，可随时获取最新答案，多次点击覆盖原有作答
2. 识别跳过标识，无法作答题型自动跳过并跳转下一题
3. 优化截图清晰度，提升AI识别准确率
4. 放开提交限制，支持连续发送题目请求
5. 适配页面路由跳转，自动重载脚本避免功能失效
6. 完善异常处理，解析报错不会中断答题流程

## 运行前置要求
必须安装并启用全部插件，缺少则无法正常使用
- Tampermonkey：脚本运行基础工具，**需授予访问网页URL、页面运行脚本权限**
- Disable Content-Security-Policy：解除网页限制，保证截图、跨页面数据通信正常
- 学习通强制粘贴助手：辅助简答题文字稳定录入

## 账号与对话准备
1. 豆包网页端提前登录个人账号
2. 新建空白对话窗口，专门用于作业答题交互，避免历史对话干扰解析结果
3. 答案格式错乱、识别异常时，可手动向豆包补发规范指令修正输出内容

## 安装步骤
1. 打开Tampermonkey管理面板，新建空白脚本
2. 粘贴完整源码，按下Ctrl+S保存即可启用
3. 确认脚本权限正常，允许跨页面读取、数据交互操作

## 使用教程
1. 分开两个浏览器标签，分别打开学习通答题页面、豆包专属答题对话页
2. 进入豆包页面，点击右下角绑定按钮，完成页面关联，绑定状态刷新后依旧保留
3. 切回学习通，点击初始化规则，统一AI答案输出格式
4. 纯文字题目选择发送文字，含图片公式题目选择发送截图
5. 切换至豆包等待AI生成答案，可自动抓取结果，也可手动点击按钮强制刷新
6. 答案自动回填对应题目，遇到无法作答内容直接跳过切换下一题
7. 勾选自动下一题，作答、跳题完成后自动切换至下一试题

## 问题排查
- 截图长时间无响应：开启CSP限制解除插件，刷新页面重试
- 答案无法自动填写：重新初始化答题规则，规范输出格式后手动抓取答案
- 页面切换后脚本失效：等待自动重载，异常情况手动刷新页面恢复
- 题目直接跳过不作答：判定为手写、画图类题型，属于正常跳题逻辑
- 点击抓取无新答案：等待AI完整回复结束后，再执行抓取操作
- AI输出格式混乱：手动在豆包发送规范答题指令，约束答案输出样式

## 现存待优化问题
1. 提交题目后需切换豆包等待答案生成，抓取完成后方可返回学习通操作
2. 大批量连续答题偶尔出现答案顺序偏差，建议单题逐一提交作答
3. 部分特殊排版题干，存在少量文字提取不全情况
